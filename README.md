# homebrew-me

Legacy and outdated app versions of casks.

## Usage

```
brew tap alswl/cask-me
brew update

brew install --cask vimr-v0.44.0
brew install --cask surge2
# etc

# untap for skip git repo update
brew untap --force alswl/cask-me
```
